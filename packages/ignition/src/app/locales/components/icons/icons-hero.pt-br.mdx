# Ícones para React e React-Native, fácil em uma solução cross-platform
Descubra como incorporar facilmente ícones populares em seus projetos React e React-Native, utilizando o RocketIcons,
um plugin não oficial do Tailwind, permitindo o compartilhamento de código entre aplicações web e nativas, fornecendo uma solução multiplataforma.
