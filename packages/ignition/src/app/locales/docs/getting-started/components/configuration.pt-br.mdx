### Configuração

Para usar <RocketIconsTextDefault /> em seus projetos, você precisa ter uma configuração mínima.  
Você precisa ter o tailwind em seu projeto e, é claro, o arquivo `tailwind.config.js`.

Nesse arquivo, importe o plugin e adicione-o ao array `plugins` assim:

```js
// tailwind.config.js
import icons from "rocketicons/tailwind";

...
plugins: [
  icons
],
...
```

Depois disso, você pode usar <RocketIconsTextDefault /> em seu projeto.
