### Installation

To install <RocketIconsTextDefault />, simply run the following command in your terminal:

```bash
npm install rocketicons
```

or

```bash
pnpm install rocketicons
```

or

```bash
yarn install rocketicons
```

or even

```bash
bun install rocketicons
```

Boom!

Now that you have installed <RocketIconsTextDefault /> in your project, you can proceed to the configuration.
