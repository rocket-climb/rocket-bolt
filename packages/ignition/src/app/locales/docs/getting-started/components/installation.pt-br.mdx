## Instalação

Para instalar <RocketIconsTextDefault />, simplesmente execute o seguinte comando no seu terminal:

```bash
npm install rocketicons
```

ou

```bash
pnpm install rocketicons
```

ou

```bash
yarn install rocketicons
```

ou até mesmo

```bash
bun install rocketicons
```

Boom!

Agora que você instalou <RocketIconsTextDefault /> no seu projeto, você pode prosseguir para a configuração.
