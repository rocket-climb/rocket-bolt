# Introdução

Bem-vindo à documentação do <RocketIconsText className="text-sky-500" />!
Este componente fornece uma coleção de ícones para uso em seus projetos React web e React Native. O RocketIcons utiliza classes do Tailwind CSS para estilização, tornando fácil personalizar e integrar em suas aplicações.


## Por quê foi criado?

<RocketIconsText className="text-slate-900 dark:text-white" /> foi criado para fornecer uma biblioteca de ícones simples e fácil de usar para desenvolvedores que desejam escrever um único código que possa funcionar tanto no **React** web quanto no **React Native**. Fomos motivados pela existência de ótimas ferramentas como [Tailwind CSS](https://tailwindcss.com/), [Expo](https://expo.dev/) e [React Icons](https://react-icons.github.io/react-icons/).
These tools are great and we love them but we wanted to create an even easier experience
 for developers working with multiple platforms with the same codebase.

The icons are designed to be easily customizable and can be used in both web
and mobile applications.

**Write once, use everywhere!**
