## Introdução

# Primeiros Passos

Bem-vindo à documentação para <RocketIconsTextDefault />!
Este componente fornece uma coleção de ícones para uso em seus projetos <SiReact className="-mt-1 icon-sky-800" /> **React** web e **React Native**.

<RocketIconsTextDefault /> utiliza classes do **Tailwind CSS** para estilização, tornando fácil a
personalização e integração em suas aplicações.

### Por que foi criado?

<RocketIconsTextDefault /> foi criado para fornecer uma biblioteca de ícones simples e fácil de
usar para desenvolvedores que desejam escrever um único código que possa funcionar tanto em
**React** web quanto em **React Native**.

Fomos motivados pela existência de ótimas
ferramentas como

<SiTailwindcss className="-mt-1" /> [Tailwind CSS](https://tailwindcss.com/),
<SiExpo className="-mt-1" /> [Expo](https://expo.dev/),
<SiReact className="-mt-1 icon-sky-500" /> [NativeWind](https://nativewind.dev) e
<SiReact className="-mt-1 icon-red-800" /> [React Icons](https://react-icons.github.io/react-icons/).
Essas ferramentas são ótimas e nós as amamos, mas queríamos criar uma experiência ainda mais fácil
para os desenvolvedores que trabalham com múltiplas plataformas com o mesmo código base.

Os ícones são projetados para serem facilmente personalizáveis e podem ser usados tanto em aplicações web
quanto móveis.

**Escreva uma vez, use em todos os lugares!**
