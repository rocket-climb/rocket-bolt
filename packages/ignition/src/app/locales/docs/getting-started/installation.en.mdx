## Installation

To install <RocketIconsText className="text-slate-900 dark:text-white" />, simply run the following command in your terminal:

```bash
npm install rocketicons
```

or

```bash
pnpm install rocketicons
```

or

```bash
yarn install rocketicons
```

or even

```bash
bun install rocketicons
```

That's it! You're ready to start using the icons in your project.
