# Adding Icons

To add an icon to your application, follow these steps:

1. Import the <RocketIconsText className="text-slate-900 dark:text-white" /> component from the RocketIcons package.

```typescript
import { RcRocketIcons } from 'rocketicons';
```

2. Use the <RocketIconsText className="text-slate-900 dark:text-white" /> component in your code, specifying the desired icon name as a prop.
