### Paleta de cores padrão

Os ícones estão disponívels em toda a paleta de cor padrão do Tailwind, usando as configurações de cores do tema.  
Siga [a documentação oficial para customizar cores](https://tailwindcss.com/docs/customizing-colors)
e adicionar sua prórpia paleta.
