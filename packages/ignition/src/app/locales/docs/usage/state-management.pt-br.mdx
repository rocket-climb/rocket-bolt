# Usando Hover, Focus e outros Estados

Como estilizar os ícones em hover, focus e muito mais
<br />
**O suporte a estados no React Native não é completo**. _Veja a documentação oficial do [NativeWind](https://www.nativewind.dev/core-concepts/states) para saber mais._
<br />
Você pode usar modificadores antes dos utilitários para condicionalmente aplicar um estilo diferente para condições desejadas.
