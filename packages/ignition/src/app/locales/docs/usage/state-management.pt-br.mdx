# Como Usar Cores e Estados

Para aplicar cores aos elementos em sua aplicação e lidar com diferentes estados como hover e focus, siga estes passos:

1. Defina valores de cor usando propriedades CSS como `color`, `background-color`, `border-color`, etc.
2. Utilize códigos de cor hexadecimal (`#RRGGBB`), valores RGB (`rgb(r, g, b)`) ou nomes de cores CSS para especificar as cores.
3. Utilize frameworks CSS como Bootstrap ou Tailwind CSS para simplificar a implementação de cores.
4. Utilize pseudo-classes como `:hover`, `:focus`, `:active`, etc., para definir estilos para diferentes estados.
5. Aplique os estilos desejados aos elementos correspondentes usando seletores CSS.
